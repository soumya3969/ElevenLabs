# Task 7: Image Resizer Tool

## Description
Resize and convert images in batch.

## Objective
Create a Python script that resizes all images in a folder.

## Tools
- Python
- Pillow (PIL)

## Deliverables
- Python script for batch image resizing
- Support for multiple image formats
- Configurable output dimensions

## Hints/Mini Guide
1. Use `os` module to read image folder
2. Use `PIL.Image` to resize and save images
3. Process images in a loop
4. Save resized images to output folder

## Outcome
Automate image processing tasks
