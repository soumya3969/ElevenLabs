# Task 3: Web Scraper for News Headlines

## Objective
Scrape top headlines from a news website.

## Tools
- Python
- requests
- BeautifulSoup

## Deliverables
- Python script
- .txt file of headlines

## Hints/Mini Guide
1. Use `requests` to fetch HTML
2. Use `BeautifulSoup` to parse `<h2>` or title tags
3. Save the titles in a `.txt` file

## Outcome
Automate data collection from a public website
