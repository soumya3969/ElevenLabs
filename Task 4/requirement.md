
# Task 4: Build a REST API with Flask

## Objective
Create a REST API that manages user data.

## Tools
- Python
- Flask
- Postman or Curl

## Deliverables
Flask app with GET/POST/PUT/DELETE routes

## Hints/Mini Guide
1. Use Flask to create endpoints
2. Store users in a dictionary or in-memory list
3. Test your API endpoints using Postman or Curl

## Outcome
API development fundamentals.

## Getting Started
1. Install Flask: `pip install flask`
2. Create your Flask application
3. Implement CRUD operations (Create, Read, Update, Delete)
4. Test your endpoints

## Example Endpoints
- `GET /users` - Retrieve all users
- `GET /users/<id>` - Retrieve a specific user
- `POST /users` - Create a new user
- `PUT /users/<id>` - Update an existing user
- `DELETE /users/<id>` - Delete a user
